# P2P мессенджер
Мессенджер в локальной сети с шифрованием.

## Возможности
- **Шифрование** — Надёжное шифрование
- **Гибкость** — Возможность использования как GUI, так и CLI версии
- **Автоматическое подключение** — Не нужно вписывать ip вручную
- **Выбор сети** — Можно выбрать нужную сеть (Локальную или виртуальную)


## 🛠️ Технологии
| Компонент | Технология |
|-----------|------------|
| **Язык** | Java 21+   |
| **Сборщик** | Maven      |
| **GUI-фреймворк** | Java FX    |


## Требования:
- Java21+
- Maven


## Установка
### Клонирование репозитория
```bash
    git clone https://github.com/prophetbrochure/p2p_messenger
```
## Сборка
```bash
    javac * или через Maven как-то там
```
## Запуск
```bash
    java src/Main
```


## 🎮 Использование
### Выбор локальной сети
Выберите локальную сеть, в которой вы будете общаться (собеседник должен быть в той же сети)
### Выбор имени
Выберите имя, под которым вас будет видеть собеседник
### Управление
1, 2, 3 - Выбор варианта
### Главное меню
При вы увидите главное меню с опциями:
```
  1
  2
  3
  4
```
